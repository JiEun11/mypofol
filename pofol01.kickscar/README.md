# pofol01 by kickscar




## outline
#### 1. backend
1. rdbms: mariadb, schema.v3
2. express mvc
3. view engine: ejs
4. model: mysql2, sql based
#### 2. frontend
1. vanila js(pure javascript)
   - dom api



## packages installed
```sh
$ npm i express
$ npm i mysql2
$ npm i ejs
$ npm i dotenv
$ npm i cross-env
$ npm i -D nodemon 
```


## project structure
<pre>

    |       |--- css

</pre>



## run application
#### 1. development
```sh
$ npm run dev
```

#### 2. production
```sh
$ npm start
```
