# pofol03.facelift


## packages installed
```sh
$ npm i express
$ npm i dotenv
$ npm i mysql2 
$ npm i ejs
$ npm i -D nodemon
```

## project structure
<pre>
/pofol03.facelift
    |--- package.json
    |--- package-lock.json
    |--- [node_modules]
    |--- public
    |       |--- js
    |       |--- css
    |       |--- images
    |--- index.js
    |--- config
    |--- routes
    |--- models      (M)
    |--- views       (V)
    |--- controllers (C)
</pre>



## run application

#### 1. development
```sh
$ npm run debug
```

#### 2. production
```sh
$ npm start
```
